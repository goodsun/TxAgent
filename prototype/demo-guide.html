🦊 MetaMask Clipboard Bridge デモガイド

⚠️ 注意：これはデモ用のプロトタイプです。実際のMetaMaskとは連携していません。

🎯 このデモで体験できること

  • ✅ トランザクション内容の日本語表示
  • ✅ 確認コードによる改ざん防止
  • ✅ クリップボード経由でのデータ受け渡し

📱 実際の利用フロー（モバイル想定）

1 通常のブラウザでDAppsを利用

Chrome や Safari で NFTマーケットやDeFiサイトを閲覧

2 トランザクション実行時

「購入」や「スワップ」ボタンを押すと確認コードが表示される

3 データをコピー

「コピーしてMetaMaskへ」ボタンでデータをクリップボードにコピー

4 MetaMaskアプリで確認

MetaMaskアプリを開いて、データを貼り付け、確認コードを照合

5 安心して承認

日本語で内容を確認してから署名

🖥️ デモを試す

DApp画面を開く → MetaMask画面を開く

※ デモでは両方をブラウザの別タブで開いて動作を確認できます

💡 ポイント

  • 確認コード：6文字の英数字で改ざんを防止
  • 日本語表示：「NFT購入」「0.1 ETH」など分かりやすく表示
  • 詐欺防止：怪しい操作は警告表示（実装予定）

🔧 技術仕様

  • HTML + JavaScript のみで実装
  • クリップボード API を使用
  • Base64 エンコードでデータ転送
  • SHA風のダイジェストで改ざん検知
